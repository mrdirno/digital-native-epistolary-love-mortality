## Entry Points for Contributors

**Start anywhere. Fork freely. No permission needed.**

### Story Entry Points

- Write a letter from Elara to Kai describing a memory she can feel but not access.
- Write a scene where Kai tries to explain a shared memory using only physical objects, no digital files.
- Write a medical report from a doctor observing a later stage of Condition Theta.
- Write a 'memory packet' from early in Kai and Elara's relationship, complete with sensory tags and emotional metadata.

### Technical Entry Points

- Design a speculative data structure for storing 'fragile' human memories in a way that prevents digital atrophy.
- Outline a protocol for a 'ground truth' verification system between two people, using a mix of digital and physical triggers.
